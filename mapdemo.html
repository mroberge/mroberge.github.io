Instructions

Points on the map represent USGS stream gauges. Click on a point to see its name and information about its current condition. The hydrograph below the map will display the previous 24 hours of stream discharge data.

Data Source:

All data are collected and provided by the US Geological Survey. For more information, visit the NWIS ( http://waterdata.usgs.gov/nwis/rt)

--The data displayed in this demo are static and hosted by Towson University, not by the original data provider (USGS)--

Description of the HydroCloud project

HydroCloud is a hydrological analysis tool consisting of a fast, lightweight mapping and analysis tool and a scalable, distributed database that works in the background. This application, which you are currently viewing in a web browser, is a partial demonstration of the interface for this project.

Martin Roberge and Michael McGuire are the principle investigators for this project, which has been generously supported with a seed grant from the TU School of Emerging Technology.
